## 高阶函数

Higher-order function

py函数和js一样都是对象

可以作为参数在函数间传递

把函数作为参数传入，这样的函数称为高阶函数，函数式编程就是指这种高度抽象的编程范式

`map`，`reduce`，`filter`，`sorted`

返回函数

```
def lazy_sum(*args):
    def sum():
        ax = 0
        for n in args:
            ax = ax + n
        return ax
    return sum
```

匿名函数

```
def build(x, y):
    return lambda: x * x + y * y
```
